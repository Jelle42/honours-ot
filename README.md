# honours-ot

Computes and visualizes optimal transport between finite collections of points in R^2 and R^3.
Some example animations can be found under /animations

sinkhorn.py - Computes unique transport matrix P given some scalar for entropy γ > 0.
visualize.py - Given a collection of points, visualize the optimal transport between those two.
geometry.py - Generate random points on the boundary of some basic shapes.
